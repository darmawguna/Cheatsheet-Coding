Semantic Tags
Kenapa Semantic?
Tag semantic memberi makna pada konten. Browser dan mesin pencari jadi lebih paham!
<main> Konten utama halaman (hanya 1!)
<section> Kelompok konten bertema sama
<footer> Info bawah halaman / kredit
<header> Bagian atas / judul area
📦
Ibarat ruangan di rumah: <main> = ruang tamu, <section> = kamar tidur, <footer> = dapur. Setiap tag punya fungsinya!

FlexboxLayout
Centering dengan Flexbox
Trik paling populer untuk menaruh kartu tepat di tengah layar!
CSS — Centering
.card-container { display: flex; justify-content: center; align-items: center; min-height: 100vh; /* Ini yang bikin kartu di tengah! */ } .card { background: #fff; border-radius: 20px; padding: 40px; width: 320px; text-align: center; box-shadow: 0 20px 60px rgba(0,0,0,.1); }
🏀
justify-content: center = lempar bola ke tengah lapangan (horizontal). align-items: center = lempar ke tengah tinggi lapangan (vertikal). Keduanya = bola persis di tengah!
CSSHover Effects
Tombol Hover Interaktif
Tombol sosmed yang berubah warna saat di-hover menggunakan transition.
CSS — Button Hover
.btn-sosmed { background: #007bff; color: white; padding: 10px 24px; border-radius: 99px; border: none; cursor: pointer; transition: all 0.3s ease; /* transition = animasi smooth */ } .btn-sosmed:hover { background: #0056cc; transform: translateY(-3px); /* naik 3px = efek "terangkat" */ }
SECTION 4

🧠 JavaScript Logic

JavaScriptCounter
Fungsi Counter Like
Ini adalah inti dari fitur interaktif — tombol Like yang angkanya bertambah!
JavaScript — Counter
// Langkah 1: Buat variabel penghitung let angkaLike = 0; // Langkah 2: Ambil elemen dari HTML const tombolLike = document.querySelector('#btn-like'); const tampilanAngka = document.querySelector('#count'); // Langkah 3: Dengarkan event "klik" tombolLike.addEventListener('click', () => { angkaLike++; // Tambah 1 setiap klik tampilanAngka.textContent = angkaLike; });
🎮
Bayangkan game dengan score counter. angkaLike = score di memori, tampilanAngka.textContent = layar score yang kamu lihat. Setiap klik = score +1!
JavaScriptAlert
Alert Saat Link Diklik
Tampilkan pesan sambutan setiap kali tombol link sosmed diklik.
JavaScript — Alert
// Ambil semua link sosmed const semuaLink = document.querySelectorAll('.btn-sosmed'); // Loop semua link, tambah event klik semuaLink.forEach(link => { link.addEventListener('click', () => { alert("Terima kasih sudah berkunjung! 🎉"); }); }); // Atau cara singkat untuk 1 tombol: document.querySelector('#btn-ig') .addEventListener('click', () => { alert("Kamu diarahkan ke Instagram!"); });
Konsep DOM3 Langkah
Rumus 3 Langkah Manipulasi DOM
Setiap kali kamu mau JS berinteraksi dengan HTML, selalu ikuti 3 langkah ini.
01
Siapkan Variabel
let counter = 0;
Variabel = wadah untuk menyimpan data/nilai
02
Ambil Elemen HTML
querySelector('#id')
querySelector = jembatan JS ke HTML
03
Tambah Event Listener
.addEventListener()
Event = reaksi terhadap aksi pengguna

🚀 Bonus Challenge

BONUSDynamic Theme
🎨 Dynamic Theme
Background berubah warna setiap Like mencapai kelipatan 5. Pakai modulo operator!
JavaScript — Dynamic Theme
const colors = [ '#ff6b6b', '#ffd93d', '#6bcb77', '#4d96ff', '#ff6bb5' ]; let colorIdx = 0; tombolLike.addEventListener('click', () => { angkaLike++; tampilanAngka.textContent = angkaLike; // % = sisa bagi (modulo) // 5 % 5 = 0, 10 % 5 = 0, dll if (angkaLike % 5 === 0) { document.body.style.background = colors[colorIdx % colors.length]; colorIdx++; } });
💡 Operator Modulo (%) = sisa bagi. Contoh: 10 % 3 = 1, karena 10 = 3×3 + sisa 1. Berguna untuk membuat sesuatu terjadi setiap N langkah!
BONUSInput Event
✍️ Real-time Name Input
Teks nama di kartu berubah secara langsung saat user mengetik! Pakai event 'input'.
HTML + JavaScript — Input Real-time
<!-- HTML: tambahkan input box --> <input type="text" id="input-nama" placeholder="Ketik namamu..."> // JavaScript: event 'input' = langsung saat ketik const inputNama = document.querySelector('#input-nama'); const displayNama = document.querySelector('#name'); inputNama.addEventListener('input', () => { // .value = teks yang diketik user displayNama.textContent = inputNama.value; }); // Bedanya 'input' vs 'change': // 'input' → langsung saat ketik (real-time) // 'change' → saat focus pindah (setelah selesai)
