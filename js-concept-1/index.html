SECTION 01 Array — Daftar dalam Satu Kotak
📋 Apa Itu Array?

Array adalah cara menyimpan banyak nilai sekaligus dalam satu variabel. Bayangkan kamu punya 30 teman sekelas — akan sangat repot kalau buat variabel satu-satu!

Analogi: Array seperti rak buku bernomor. Setiap buku punya nomor urut (index), dimulai dari nomor 0 — bukan 1! Jadi buku pertama ada di nomor 0.
[0]
"Andi"
[1]
"Budi"
[2]
"Citra"
[3]
"Dina"
[4]
"Eko"
JavaScript
const teman = ["Andi", "Budi", "Citra", "Dina", "Eko"]; // Akses berdasarkan index (mulai dari 0!) console.log(teman[0]); // → "Andi" console.log(teman[2]); // → "Citra" // Cek jumlah item console.log(teman.length); // → 5 // Ambil item terakhir console.log(teman[teman.length - 1]); // → "Eko"
⚠️ Ingat! Index array dimulai dari 0, bukan 1. Jadi item ke-1 ada di index 0, item ke-2 ada di index 1, dan seterusnya!
🔧 Metode Array yang Wajib Tahu
.push()
Tambah item ke akhir array
teman.push("Fajar");
// → ["Andi",...,"Fajar"]
.pop()
Hapus item paling akhir
teman.pop();
// → "Eko" dihapus
.includes()
Cek apakah nilai ada di array
teman.includes("Budi");
// → true
.indexOf()
Cari index dari nilai tertentu
teman.indexOf("Citra");
// → 2
.join()
Gabung semua item jadi satu string
teman.join(", ");
// → "Andi, Budi, ..."
.length
Hitung jumlah item (bukan metode, tapi property)
teman.length;
// → 5

SECTION 02 Looping — Kerja Berulang Otomatis
🔁 Mengapa Butuh Loop?

Bayangkan kamu punya 100 siswa dan ingin mencetak nama mereka satu per satu. Tanpa loop, kamu harus tulis console.log() sebanyak 100 kali. Dengan loop, cukup tulis sekali!

Analogi: Loop seperti mesin fotokopi. Kamu tidak perlu menekan tombol 100 kali — cukup atur jumlahnya, lalu biarkan mesin bekerja sendiri.

❌ Tanpa Loop (tidak efisien)

JS
console.log(teman[0]); // Andi console.log(teman[1]); // Budi console.log(teman[2]); // Citra // ...terus sampai 100x 😩

✅ Dengan Loop (efisien!)

JS
teman.forEach(function(nama) { console.log(nama); }); // Otomatis cetak semua! 🎉

.forEach() — Paling mudah untuk array!

forEach adalah cara paling simpel untuk memproses setiap item dalam array. Cocok untuk pemula dan dipakai di mana saja.

const buah = ["apel", "mangga", "jeruk"];

buah.forEach(function(item) {
console.log(item);
});
// → apel
// → mangga
// → jeruk

SECTION 03 String Methods — Manipulasi Teks
✂️ Teks Bisa Dimanipulasi!

String (teks) punya banyak metode bawaan untuk diubah, dipotong, dicari, dan digabung. Ini sangat berguna saat kamu perlu memproses input dari user!

Analogi: String seperti kalimat di secarik kertas. Kamu bisa menggunting bagian tertentu, menghapus spasi, mengubah huruf besar-kecil, atau mengganti kata tertentu — semuanya tanpa mengubah kertas aslinya!
JavaScript
const kata = " Halo Dunia! "; kata.toUpperCase(); // → " HALO DUNIA! " kata.toLowerCase(); // → " halo dunia! " kata.trim(); // → "Halo Dunia!" (hapus spasi tepi) kata.length; // → 15 (termasuk spasi!) kata.includes("Dunia"); // → true kata.replace("Dunia", "Indonesia"); // → " Halo Indonesia! " kata.slice(2, 6); // → "Halo"

SECTION 04 Form Handling — Ambil Data dari User
📝 Mengambil Nilai dari Form

Hampir semua website punya form — login, registrasi, pencarian. JavaScript digunakan untuk mengambil apa yang diketik user dan melakukan sesuatu dengannya.

Analogi: Form seperti lembar ujian. User mengisi jawaban, lalu saat tombol submit ditekan, JavaScript berperan sebagai guru yang memeriksa dan memproses jawaban tersebut.
HTML + JavaScript
<!-- HTML --> <input type="text" id="namaInput"> <button id="btnKirim">Kirim</button> <p id="hasilnya"></p> // JavaScript const input = document.getElementById('namaInput'); const tombol = document.getElementById('btnKirim'); const hasil = document.getElementById('hasilnya'); tombol.addEventListener('click', function() { const nama = input.value; // .value = ambil isi input! hasil.textContent = "Halo, " + nama + "!"; });
💡 Kunci Utama: Untuk mengambil nilai dari input, gunakan .value — bukan .textContent. .value khusus untuk elemen form seperti input, textarea, dan select.

SECTION 05 Validasi Form — Form yang Pintar
🛡️ Kenapa Harus Validasi?

Validasi adalah pengecekan data sebelum diproses. Tanpa validasi, user bisa mengosongkan form, mengirim email yang tidak valid, atau memasukkan angka negatif untuk umur!

Analogi: Validasi seperti satpam di pintu masuk. Sebelum boleh masuk, dicek dulu: "Ada KTP? Alamat benar? Sudah isi semua kolom?" — baru boleh lanjut!
JavaScript
function validasiForm() { const nama = document.getElementById('nama').value.trim(); const email = document.getElementById('email').value.trim(); const umur = document.getElementById('umur').value; // ✅ Cek apakah nama kosong if (nama === "") { alert("Nama tidak boleh kosong!"); return; // hentikan fungsi di sini } // ✅ Cek apakah email mengandung "@" if (!email.includes("@")) { alert("Format email tidak valid!"); return; } // ✅ Cek apakah umur masuk akal if (umur < 5 || umur > 100) { alert("Umur tidak valid!"); return; } alert("Form berhasil dikirim! 🎉"); }
