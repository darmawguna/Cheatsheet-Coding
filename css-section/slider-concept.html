SECTION 1 Apa Itu Auto Slider?
Definisi & Konsep Dasar

Auto Slider (atau Image Carousel) adalah komponen UI yang menampilkan beberapa gambar secara bergiliran dan berpindah otomatis tanpa interaksi user. Sering dipakai di banner website, portfolio, dan landing page.

🎠
Analogi: Bayangkan slide foto di presentasi PowerPoint yang otomatis berpindah setiap beberapa detik — itulah auto slider! Bedanya, ini dibuat dengan kode HTML/CSS/JS dan hidup di browser.
UI Component setInterval CSS Transform DOM Manipulation

SECTION 2 Struktur HTML
Kerangka HTML Slider

Slider membutuhkan 3 elemen utama dalam strukturnya: container pembungkus, wrapper slide (yang bergerak), dan setiap slide individu.

HTML
<!-- 1. Container: batas tampilan (overflow: hidden) --> <div class="slider-container"> <!-- 2. Wrapper: bergerak ke kiri/kanan --> <div class="slides" id="slide-wrapper"> <!-- 3. Setiap slide (min-width: 100%) --> <div class="slide"> <img src="gambar1.jpg" alt="Slide 1"> </div> <div class="slide"> <img src="gambar2.jpg" alt="Slide 2"> </div> </div> <!-- 4. Tombol navigasi panah --> <button class="prev" onclick="changeSlide(-1)">❮</button> <button class="next" onclick="changeSlide(1)">❯</button> <!-- 5. Dots indikator posisi --> <div class="dots-container"> <span class="dot active" onclick="currentSlide(0)"></span> <span class="dot" onclick="currentSlide(1)"></span> </div> </div>
🎬
Analogi: Bayangkan .slider-container sebagai layar bioskop (bingkai tetap), .slides sebagai gulungan film yang bisa digeser, dan setiap .slide sebagai satu frame film.

SECTION 3 Styling CSS
CSS Inti — Container & Overflow

Kunci utama slider ada di dua properti: overflow: hidden pada container (menyembunyikan slide di luar), dan display: flex pada wrapper (membuat slide berjajar horizontal).

CSS
/* Container: batas pandang, sembunyikan yang di luar */ .slider-container { position: relative; width: 90%; max-width: 800px; overflow: hidden; /* ← KUNCI! Sembunyikan slide lain */ border-radius: 15px; box-shadow: 0 10px 25px rgba(0,0,0,.2); } /* Wrapper slide: flex = semua slide berjajar horizontal */ .slides { display: flex; /* ← Berjajar ke kanan */ transition: transform 0.6s cubic-bezier(.4,0,.2,1); width: 100%; } /* Setiap slide: lebar 100% dari container */ .slide { min-width: 100%; /* ← Setiap slide selebar container */ position: relative; } .slide img { width: 100%; display: block; object-fit: cover; }
CSS Tombol Panah
CSS
.prev, .next { position: absolute; top: 50%; transform: translateY(-50%); background: rgba(255,255,255,.7); border: none; padding: 15px; cursor: pointer; border-radius: 50%; z-index: 10; transition: 0.3s; } .prev { left: 20px; } .next { right: 20px; }
CSS Dots Indikator
CSS
.dots-container { position: absolute; bottom: 20px; left: 50%; transform: translateX(-50%); display: flex; gap: 10px; } .dot { width: 12px; height: 12px; background: rgba(255,255,255,.5); border-radius: 50%; cursor: pointer; transition: 0.3s; } .dot.active { background: white; transform: scale(1.3); }

SECTION 4 JavaScript — Logika Slider
Fungsi showSlide() — Inti Pergerakan

Fungsi ini menggeser wrapper menggunakan translateX() berdasarkan index slide yang aktif, lalu memperbarui dot yang menyala.

JavaScript
var slides = document.getElementById('slide-wrapper'); var dots = document.querySelectorAll('.dot'); var totalSlides = document.querySelectorAll('.slide').length; var index = 0; var slideInterval; function showSlide(n) { index = n; // Loop balik jika index melewati batas if (index >= totalSlides) index = 0; if (index < 0) index = totalSlides - 1; // Geser wrapper: -index * 100% = geser ke slide ke-N slides.style.transform = 'translateX(' + (-index * 100) + '%)'; // Update dots: hapus semua .active, beri ke dot saat ini for (var i = 0; i < dots.length; i++) { dots[i].classList.remove('active'); } dots[index].classList.add('active'); }
📐
Rumus translateX: Jika ada 3 slide dan kita mau ke slide ke-2 (index=1), maka translateX(-1 × 100%) = translateX(-100%) — artinya geser wrapper 100% ke kiri!
Navigasi Manual
JavaScript
// Tombol panah: maju/mundur 1 step function changeSlide(step) { resetTimer(); showSlide(index + step); } // Klik dot: langsung ke slide tertentu function currentSlide(n) { resetTimer(); showSlide(n); } // changeSlide(1) → maju ke depan // changeSlide(-1) → mundur ke belakang
Auto Slide & Timer
JavaScript
// Mulai timer otomatis dengan setInterval function startAutoSlide() { slideInterval = setInterval(function() { showSlide(index + 1); }, 4000); // setiap 4000ms = 4 detik } // Reset timer saat user klik manual function resetTimer() { clearInterval(slideInterval); startAutoSlide(); } // Jalankan pertama kali saat halaman dimuat startAutoSlide();
📊 Alur Kerja Lengkap JavaScript
1
Inisialisasi variabel
index = 0 (slide aktif), ambil referensi elemen DOM, hitung total slide
2
startAutoSlide() dipanggil
setInterval berjalan, setiap N detik memanggil showSlide(index + 1)
3
showSlide(n) menggeser slider
Validasi index (loop balik jika melebihi), terapkan translateX, update dot
4
User klik panah/dot → resetTimer()
clearInterval menghentikan timer lama, lalu startAutoSlide memulai ulang dari 0
