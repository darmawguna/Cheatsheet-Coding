⚡ Function Challenge — Week 3 & 4

Mesin Sapa
Otomatis 🤖

Buat sebuah fungsi JavaScript yang bisa menyapa siapa saja, kapan saja — secara otomatis dan personal!

🟡 Tingkat: Pemula
⏱️ Estimasi: 30–60 menit
🎯 Skill: Function + if/else
BRIEFING Apa yang Harus Dibuat?
📋 Deskripsi Challenge

Kamu diminta membuat sebuah function JavaScript bernama sapa() yang menerima input dari user dan menghasilkan sapaan yang berbeda-beda tergantung kondisi.

🎯 Tujuan Utama: Latihan membuat function yang menerima parameter, menggunakan if/else untuk logika kondisi, dan menghubungkannya ke DOM sehingga hasilnya tampil di halaman.

📥 Input yang Diterima:

  • nama Nama pengguna (string)
  • waktu Pagi / Siang / Malam
  • mood Semangat / Santai / Ngantuk

📤 Output yang Dihasilkan:

  • 👋 Sapaan yang berubah sesuai waktu
  • 💬 Pesan motivasi sesuai mood
  • 🎨 Tampil di halaman (DOM)

CHALLENGE Soal Bertingkat
📌 Cara mengerjakan: Kerjakan dari Level 1 dulu. Setelah berhasil, lanjut ke Level 2, dst. Setiap level membangun di atas level sebelumnya — jangan skip!
✅ Level 1 Function dasar dengan 1 parameter
  • 01
    Buat function bernama sapa(nama)
    Function ini menerima satu parameter: nama (string). Gunakan function keyword untuk membuatnya.
    💡 Ingat: function namaFungsi(parameter) { ... }
  • 02
    Return kalimat sapaan sederhana
    Fungsi harus mengembalikan string: "Halo, [nama]! Selamat datang." Gunakan return dan gabungkan string dengan + atau template literal.
    💡 return "Halo, " + nama + "!" atau `Halo, ${nama}!`
  • 03
    Hubungkan ke HTML — tampilkan di halaman
    Ambil nilai dari <input id="inputNama">, panggil function, dan tampilkan hasilnya di <p id="hasil"> menggunakan .textContent.
    💡 document.getElementById('hasil').textContent = sapa(nama);

Expected Output Level 1:

// saat user mengetik "Budi" lalu klik tombol: Halo, Budi! Selamat datang. // saat user mengetik "Siti": Halo, Siti! Selamat datang.
Struktur function untuk Level 1:

1. Buat function dengan keyword function, diikuti nama fungsi dan parameter dalam kurung.
2. Di dalam kurung kurawal { }, tulis return diikuti string sapaan.
3. Untuk menggabungkan teks dan variabel, gunakan tanda + atau backtick template literal `Halo, ${nama}!`.
4. Panggil fungsi dari event listener tombol, simpan hasilnya, lalu masukkan ke .textContent elemen HTML.
⭐ Level 2 Tambah parameter waktu + logika if/else
  • 01
    Tambah parameter waktu ke function
    Ubah function menjadi sapa(nama, waktu). Parameter waktu menerima string: "pagi", "siang", atau "malam".
  • 02
    Buat sapaan yang berubah sesuai waktu
    Gunakan if / else if / else untuk membuat sapaan yang berbeda:
    • "pagi" → "Selamat pagi, [nama]! ☀️ Semangat ya!"
    • "siang" → "Selamat siang, [nama]! 🌤️ Jangan lupa makan siang!"
    • "malam" → "Selamat malam, [nama]! 🌙 Istirahat yang cukup ya!"
    💡 Gunakan: if (waktu === "pagi") { return "..." }
  • 03
    Tambahkan dropdown waktu ke HTML
    Tambahkan <select id="pilihWaktu"> dengan 3 opsi. Ambil nilainya dengan .value dan kirim ke function.
    💡 const waktu = document.getElementById('pilihWaktu').value;
  • 04
    Validasi — jika nama kosong, jangan sapa!
    Sebelum memanggil function, cek dulu apakah input nama tidak kosong. Jika kosong, tampilkan pesan error: "Hei, siapa nama kamu? 👀"
    💡 if (nama.trim() === "") { ... }

Expected Output Level 2:

// nama: "Andi", waktu: "pagi" Selamat pagi, Andi! ☀️ Semangat ya! // nama: "Citra", waktu: "malam" Selamat malam, Citra! 🌙 Istirahat yang cukup ya! // nama kosong: Hei, siapa nama kamu? 👀
Struktur if/else untuk waktu:

Gunakan === (triple equals) untuk perbandingan string yang tepat. Urutan yang disarankan:

if (waktu === "pagi") { ... }
else if (waktu === "siang") { ... }
else { ... } ← ini untuk malam atau kondisi lainnya

Untuk validasi nama kosong: lakukan pengecekan di luar function, sebelum memanggil sapa(). Gunakan .trim() agar spasi-spasi saja juga dianggap kosong.
🔥 Level 3 Full feature + parameter mood
  • 01
    Tambah parameter ketiga: mood
    Function jadi sapa(nama, waktu, mood). Parameter mood menerima: "semangat", "santai", atau "ngantuk".
  • 02
    Tambah pesan motivasi berdasarkan mood
    Buat pesan tambahan (terpisah dari sapaan waktu):
    • "semangat" → "Woah, energimu menular! 🔥 Ayo kita produktif hari ini!"
    • "santai" → "Santai itu perlu. Tapi jangan lupa tugas ya! 😄"
    • "ngantuk" → "Ngantuk? Minum kopi dulu atau splash air ke muka! 😂"
  • 03
    Return dua baris — sapaan + motivasi
    Function harus mengembalikan kedua string. Tampilkan sapaan di satu elemen, dan pesan motivasi di elemen lain (gunakan 2 elemen HTML berbeda).
    💡 Bisa return object: return { sapaan: "...", motivasi: "..." }
  • 04
    Tampilkan emoji yang berubah sesuai waktu di halaman
    Tambahkan elemen <span id="emojiDisplay"> yang menampilkan emoji berbeda: ☀️ untuk pagi, 🌤️ untuk siang, 🌙 untuk malam — diubah via .textContent.
  • 05
    🏆 BONUS: Hitung berapa kali tombol sudah diklik
    Buat variabel let hitungSapa = 0 di luar function. Setiap kali tombol diklik, tambahkan 1. Tampilkan: "Kamu sudah disapa [N] kali!"
    💡 hitungSapa++; lalu tampilkan nilainya

Expected Output Level 3:

// nama: "Dina", waktu: "pagi", mood: "semangat" 🌅 Selamat pagi, Dina! ☀️ Semangat ya! 💬 Woah, energimu menular! 🔥 Ayo kita produktif hari ini! // nama: "Eko", waktu: "malam", mood: "ngantuk" 🌙 Selamat malam, Eko! 🌙 Istirahat yang cukup ya! 💬 Ngantuk? Minum kopi dulu atau splash air ke muka! 😂 Kamu sudah disapa 3 kali!
Tips mengembalikan dua nilai dari satu function:

Cara paling mudah: gunakan object sebagai return value.

function sapa(nama, waktu, mood) {
  let sapaan = "";
  let motivasi = "";
  if (waktu === "pagi") { sapaan = "Selamat pagi..."; }
  if (mood === "semangat") { motivasi = "Woah..."; }
  return { sapaan, motivasi }; // return object!
}

Lalu di luar:
const hasil = sapa(nama, waktu, mood);
document.getElementById('x').textContent = hasil.sapaan;
document.getElementById('y').textContent = hasil.motivasi;

// ── Function utama ────────────────────────── function sapa(nama, waktu, mood) { // Tentukan sapaan berdasarkan waktu let sapaan = ""; if (waktu === "pagi") { sapaan = `Selamat pagi, ${nama}! ☀️ Semangat ya!`; } else if (waktu === "siang") { sapaan = `Selamat siang, ${nama}! 🌤️ Jangan lupa makan siang!`; } else { sapaan = `Selamat malam, ${nama}! 🌙 Istirahat yang cukup ya!`; } // Tentukan pesan motivasi berdasarkan mood let motivasi = ""; if (mood === "semangat") { motivasi = "Woah, energimu menular! 🔥 Ayo kita produktif hari ini!"; } else if (mood === "santai") { motivasi = "Santai itu perlu. Tapi jangan lupa tugas ya! 😄"; } else { motivasi = "Ngantuk? Minum kopi dulu atau splash air ke muka! 😂"; } return { sapaan, motivasi }; } // ── Counter ───────────────────────────────── let hitungSapa = 0; // ── Event Listener ─────────────────────────── document.getElementById('btnSapa').addEventListener('click', function() { const nama = document.getElementById('inputNama').value.trim(); const waktu = document.getElementById('pilihWaktu').value; const mood = document.getElementById('pilihMood').value; // Validasi if (nama === "") { document.getElementById('hasil').textContent = "Hei, siapa nama kamu? 👀"; document.getElementById('hasil').style.display = "block"; document.getElementById('motivasi').style.display = "none"; return; } if (!waktu || !mood) { document.getElementById('hasil').textContent = "Pilih waktu dan mood dulu ya! 😊"; document.getElementById('hasil').style.display = "block"; return; } // Panggil function const hasilSapa = sapa(nama, waktu, mood); // Tampilkan ke DOM const elHasil = document.getElementById('hasil'); const elMotivasi = document.getElementById('motivasi'); const elHitung = document.getElementById('hitungLabel'); elHasil.textContent = hasilSapa.sapaan; elMotivasi.textContent = hasilSapa.motivasi; elHasil.style.display = "block"; elMotivasi.style.display = "block"; hitungSapa++; elHitung.textContent = `Kamu sudah disapa ${hitungSapa} kali!`; });
