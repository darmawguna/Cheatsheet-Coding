🏗️ The Layout Engine

🪟 1. Display Property
Setiap tag HTML punya "sifat bawaan" dalam mengambil ruang. Klik kartu di bawah untuk lihat contohnya!
display: block
🚌 Block — Si Rakus Baris
Selalu ambil satu baris penuh, bisa diatur lebar & tinggi.
divh1psection
display: inline
🚶 Inline — Ngikut Teks
Ikut aliran teks, lebar sesuai isi, TIDAK bisa diatur lebar/tinggi.
spanastrong
display: inline-block
🏍️ Inline-Block — Jalan Bareng tapi Bisa Diatur
Berjajar ke samping seperti inline, tapi bisa diatur ukurannya seperti block.
imgbutton
display: none
👻 None — Hilang Dari Dunia
Menghilangkan elemen sepenuhnya — tidak terlihat, tidak makan tempat.
menu dropdownmodal tersembunyi
▶ Klik salah satu kartu di atas untuk melihat preview
CSS 
/* Contoh penggunaan Display Property */
.kotak-block    { display: block; }
.teks-inline    { display: inline; }
.button-custom  { display: inline-block; }
.menu-sembunyi  { display: none; }

📍 3. CSS Positioning
Klik setiap tipe positioning untuk melihat cara kerjanya secara visual.
🪑
relative
Tetap di posisi asli, tapi bisa digeser pakai top/left/right/bottom. Parent lain tidak terpengaruh.
🏃
absolute
Melayang bebas relatif ke parent yang ber-position: relative. Keluar dari aliran normal.
🕶️
fixed
Menempel di layar. Tidak peduli scroll, posisinya selalu sama di viewport.
parent (relative)
child (relative)
CSS — relative 
/* Trik: relative di parent, absolute di child */
.parent { position: relative; }

.badge {
  position: relative;
  /* Geser dari posisi asli */
  top: 0px; right: 0px;
}
⚡ Kapan Pakai Apa?
relative
Buat "anchor" untuk anak yang absolute, atau geser elemen sedikit.
absolute
Badge, tooltip, dropdown, icon di atas gambar.
fixed
Navbar yang menempel saat scroll, tombol "back to top", chat widget.
sticky
Header tabel yang ikut sampai batas tertentu lalu nempel.
SECTION 4

📱 Responsive & Interactivity

📐 4. Relative Units — Geser Slider, Lihat Bedanya!
Pakai satuan relatif agar website bisa "bernafas" di semua ukuran layar.
width: 50%
0 100 50%
% — Persentase dari Parent
Kalau parent lebarnya 400px, maka width: 50% = 200px. Berguna untuk layout kolom-kolom yang proporsional.
%
Relatif ke parent. Cocok untuk lebar kolom.
rem
Relatif ke root font (1rem=16px). Cocok untuk font-size.
vw
Relatif ke lebar layar. 100vw = layar penuh.
vh
Relatif ke tinggi layar. 100vh = tinggi penuh.
🦎 5. Media Queries — Simulasi Responsif
Klik ikon perangkat untuk melihat bagaimana layout berubah di tiap ukuran layar.
< 480px
🌐 MySite
☰
Menampilkan: Mobile (< 480px) — Sidebar disembunyikan, grid 1 kolom, hamburger menu aktif
CSS — Mobile First Strategy 
/* ① TULIS UNTUK HP DULU (tanpa @media) */
.sidebar       { display: none; }
.grid-container{ grid-template-columns: 1fr; }
.hamburger     { display: block; }
.nav-links     { display: none; }

/* ② TABLET (≥ 768px) */
@media (min-width: 768px) {
  .sidebar        { display: block; }
  .grid-container { grid-template-columns: repeat(2, 1fr); }
}

/* ③ DESKTOP (≥ 1024px) */
@media (min-width: 1024px) {
  .hamburger      { display: none; }
  .nav-links      { display: flex; }
  .grid-container { grid-template-columns: repeat(3, 1fr); }
}
✨ 6. Transitions & Hover Effects
Hover setiap kartu untuk merasakan efek animasi yang berbeda-beda!
📦
Scale (Membesar)
Elemen mengecil/membesar saat hover
transform: scale(1.1)
← hover sini!
🎨
Color Change
Background berubah warna secara halus
transition: background .4s
← hover sini!
🚀
Slide Up
Elemen bergerak naik saat hover
transform: translateY(-8px)
← hover sini!
🌑
Shadow Lift
Bayangan muncul, berasa mengangkat
box-shadow + transform
← hover sini!
🔄
Rotate
Elemen berputar sedikit saat hover
transform: rotate(15deg)
← hover sini!
👁️
Fade
Transparansi berubah secara halus
transition: opacity .3s
← hover sini!
CSS — Template Transition Lengkap 
/* SELALU pasang transition di elemen NORMAL (bukan :hover) */
.kartu {
  transition: all 0.3s ease;
  /*         ↑      ↑       ↑
          apa   durasi   timing */
}

.kartu:hover {
  transform: translateY(-4px) scale(1.02);
  box-shadow: 0 20px 40px rgba(0,0,0,0.3);
}

/* Timing Functions: */
/* ease         → mulai cepat, akhir melambat (default, paling natural) */
/* ease-in-out  → lambat-cepat-lambat (elegan) */
/* linear       → konstan (untuk loading bar, animasi berputar) */
/* cubic-bezier → custom timing sesuai keinginan */
